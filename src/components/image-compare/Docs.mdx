## Keyboard Support

### Tab

- Moves focus to the component

### ArrowRight / ArrowDown

- Increments the value

### ArrowLeft / ArrowUp

- Decrements the value

### Home

- Set the minimum value

### End

- Set the maximum value

### Page Up

- Increments the value by 10 steps

### Page Down

- Decrements the value by 10 steps
